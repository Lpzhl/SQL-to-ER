# 贡献者行为准则 | Contributor Covenant Code of Conduct

## 我们的承诺 | Our Pledge

在互动和协作的利益中，我们承诺为参与 SQL-to-ER 项目和社区的所有人，无论其年龄、体型、可见或不可见的残疾、种族、族裔、性征、性身份和表达、经验水平、教育程度、社会经济地位、国籍、个人外表、种族、宗教或性身份和取向如何，都提供一个无骚扰的环境。

## 我们的标准 | Our Standards

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 对不同的观点和经验保持尊重
- 友好地接受建设性批评
- 专注于对社区最有利的东西
- 对其他社区成员表示同情

不可接受的行为包括：

- 性化的语言或图像和不受欢迎的性关注或进展
- 骚扰、侮辱或贬低评论以及个人或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的隐私信息，例如物理地址或电子邮件地址
- 其他在专业环境中可能被合理认为不适当的行为

## 执法 | Enforcement

项目维护者负责阐明可接受的行为标准，并应对不可接受的行为采取适当的纠正措施。

项目维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，或因其他行为被认为不适当、威胁、冒犯或有害而临时或永久禁止任何贡献者。

## 报告 | Reporting

如果您遭遇或看到不可接受的行为，请通过以下方式报告问题：

- 通过 GitHub Issues 提交报告
- 发送电子邮件至 [maintainers@example.com](mailto:maintainers@example.com)

所有投诉将进行审查和调查，并将根据需要进行适当回应。
